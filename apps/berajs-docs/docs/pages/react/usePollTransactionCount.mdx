# usePollTransactionCount

React hook for polling the transaction count for a given address.

## Import

```ts
import { usePollTransactionCount } from "@bera/berajs";
```

## Usage

```ts twoslash [example.ts]
import { usePollTransactionCount } from "@bera/berajs";

const exampleAddress = "0x1..";

const { data: txnCount, refresh } = usePollTransactionCount({
  address: exampleAddress,
});
```

:::
