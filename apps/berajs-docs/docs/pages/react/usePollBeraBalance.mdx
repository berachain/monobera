# usePollBeraBalance

React hook to retrieve the current Bera balance of the connected account.

## Import

```ts twoslash
import { usePollBeraBalance } from "@bera/berajs";
```

## Usage

```ts twoslash [example.ts]
import { usePollBeraBalance } from "@bera/berajs";

const { data: beraBalance, refresh } = usePollBeraBalance({
  address: "0x...",
});
```
