# usePollHoneyPreview

React hook to fetching the preview of the swap amount for various transaction methods involving the "honey" token.

## Initialization

Ensure you have followed the steps from [React > Getting Started](/react/getting-started) before getting started

## Import

```ts
import { usePollHoneyPreview } from "@bera/berajs";
```

## Usage

```ts
const { data: honeyPreview } = usePollHoneyPreview({
  collateral,
  amount,
  mint: true,
  given_in: true,
});
```

## Parameters

#### Call signature

```ts
function usePollHoneyPreview({
  { collateral, amount, mint, given_in }: UsePollHoneyPreviewArgs,
  options?: DefaultHookOptions,
});
```

#### args: `UsePollHoneyPreviewArgs`

| Name         | Type                     | Description                                                 | Required |
| ------------ | :----------------------- | :---------------------------------------------------------- | :------: |
| `collateral` | [`Token`](/types/Token/) | The token used as collateral in the transaction.            |  `true`  |
| `amount`     | `string`                 | The amount of the collateral or currency involved.          |  `true`  |
| `mint`       | `boolean`                | Indicates whether the operation is minting.                 |  `true`  |
| `given_in`   | `boolean`                | Specifies if the amount is given as input for calculations. |  `true`  |

#### options?: `DefaultHookOptions`

See type [`DefaultHookOptions`](/types/DefaultHookOptions/)

| Name      | type                                               | Description                            | Required |
| --------- | :------------------------------------------------- | :------------------------------------- | :------: |
| `options` | [`DefaultHookOptions`](/types/DefaultHookOptions/) | options and configuration for the hook | `false`  |

## Returns

| Name           | Type                         | Description                                                                | Optional |
| -------------- | :--------------------------- | :------------------------------------------------------------------------- | :------: |
| `data`         | `string`                     | The computed amount resulting from the operation, formatted as a string.   | `false`  |
| `error`        | `any \| undefined`           | any errors while fetching the data                                         |  `true`  |
| `mutate`       | `(newData?: string) => void` | swr mutate to refetch or manually update the data in cache if arg provided |  `true`  |
| `refetch`      | `() => void`                 | swr mutate to refetch the data                                             |  `true`  |
| `isValidating` | `boolean`                    | boolean flag representing whether the hook is currently validating data    | `false`  |
| `isLoading`    | `boolean`                    | boolean flag representing whether the hook is currently loading data       | `false`  |
