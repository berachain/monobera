# usePollReservesDataList

React SWR hook for lend global reserved data.

## Initialization

Ensure you have followed the steps from [React > Getting Started](/react/getting-started) before getting started

## Import

```ts
import { usePollReservesDataList } from "@bera/berajs";
```

## Usage

```ts
// Poll global reserves data
const {
  data: collateralRatesDictionary,
  formattedReserves,
  baseCurrencyData,
  totalBorrowed,
  marketSize,
  getSelectedReserve,
} = usePollReservesDataList();

// get selected reserve info
const collateralRate = getSelectedReserve(address);
```

## Parameters

## Returns

| Property             | Type                                                                      | Description                                                                 | Optional |
| -------------------- | ------------------------------------------------------------------------- | --------------------------------------------------------------------------- | :------: |
| `data`               | `{formattedReserves: ReserveData[]; baseCurrencyData: BaseCurrencyData;}` | Function to retrieve a specific `ReserveData` by its address.               | `false`  |
| `getSelectedReserve` | `(address: Address) => ReserveData \| undefined`                          | Function to retrieve a specific `ReserveData` by its address.               | `false`  |
| `formattedReserves`  | [`ReserveData[]`](/types/ReserveData/)                                    | Array of `ReserveData`, each formatted for easy display.                    | `false`  |
| `baseCurrencyData`   | [`BaseCurrencyData \| undefined`](/types/BaseCurrencyData/)               | Data about the base currency used in the reserve calculations.              | `false`  |
| `totalBorrowed`      | `number`                                                                  | Total amount borrowed from all reserves, typically summed in base currency. | `false`  |
| `marketSize`         | `number`                                                                  | Total market size of all reserves, often represented in base currency.      | `false`  |
| `error`              | `any \| undefined`                                                        | any errors while fetching the data                                          |  `true`  |
| `mutate`             | `(newData?: string) => void`                                              | swr mutate to refetch or manually update the data in cache if arg provided  |  `true`  |
| `refetch`            | `() => void`                                                              | swr mutate to refetch the data                                              |  `true`  |
| `isValidating`       | `boolean`                                                                 | boolean flag representing whether the hook is currently validating data     | `false`  |
| `isLoading`          | `boolean`                                                                 | boolean flag representing whether the hook is currently loading data        | `false`  |

See Type [`ReserveData`](/types/ReserveData/)

See Type [`BaseCurrencyData`](/types/BaseCurrencyData/)
