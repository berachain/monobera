# useTokens

React hook for providing an interface to a [token list](https://tokenlists.org/) compatible json file. For berachain, default token lists are maintained [here](https://github.com/berachain/default-token-list).

`useTokens` allows for management of custom tokens to be added and removed in browser localStorage.

## Import

```ts
import { useTokens } from "@bera/berajs";
```

## Required config

For more information on BeraConfig, see [BeraConfig](/bera-config/)
```ts twoslash
import { type BeraConfig } from "@bera/berajs";
export const beraConfig: BeraConfig = {
  endpoints: {
    tokenList: "https://..",
  },
};
```

## Usage

```ts twoslash [example.ts]
import { useTokens, type Token } from "@bera/berajs";

const {
  data: tokens,
  addNewToken,
  removeToken,
} = useTokens();

const newToken: Token = {
  address: "0x...",
  name: "...",
  symbol: "...",
  decimals: 18,
};

addNewToken(newToken);
removeToken(newToken);
```

## Parameters

#### options: `DefaultHookOptions`

| Name      | type                                               | Description                            | Required |
| --------- | :------------------------------------------------- | :------------------------------------- | :------: |
| `options` | [`DefaultHookOptions`](/types/DefaultHookOptions/) | options and configuration for the hook | `false`  |


## Returns `IUseTokens`


| Name           | type                                | Description                                                                | Optional |
| -------------- | :---------------------------------- | :------------------------------------------------------------------------- | :------: |
| `data`         | `GetTokens`        | user's wallet balance data                                           |  `true`  |
| `error`        | `any \| undefined`                  | any errors while fetching the data                                         |  `true`  |
| `mutate`       | `() => void` | swr mutate to refetch or manually update the data in cache if arg provided |  `true`  |
| `refetch`      | `() => void`                        | swr mutate to refetch the data                                             |  `true`  |
| `isValidating` | `boolean`                           | boolean flag representing whether the hook is currently validating data    | `false`  |
| `isLoading`    | `boolean`                           | boolean flag representing whether the hook is currently loading data       | `false`  |
| `addNewToken`    | `(token: Token) => void`                   | function for adding custom tokens to the list using localStorage     | `false`  |
| `removeToken`    | `(token: Token) => void`                         |  function for removing custom tokens to the list using localStorage   | `false`  |


See type [`GetTokens`](/types/GetTokens/)
