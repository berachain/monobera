# usePoolRecentSwaps

Hook to fetch a list of recent swaps for a given pool.

## Initialization

Ensure you have followed the steps from [React > Getting Started](/react/getting-started) before getting started

## Import

```ts
import { usePoolRecentSwaps } from "@bera/berajs";
```

## Usage

```ts
import { usePoolRecentSwaps } from "@bera/berajs";

const pool: PoolV2 = ...

function App() {
    const { data: recentSwaps, isLoading } = usePoolRecentSwaps({ pool });

    ...
};
```
