# usePoolRecentProvisions

React SWR hook to fetch a list of recent provisions for a given pool. Provisions are pool liquidity add and withdraw events.

## Initialization

Ensure you have followed the steps from [React > Getting Started](/react/getting-started) before getting started

## Import

```ts
import { usePoolRecentProvisions } from "@bera/berajs";
```

## Usage

```ts
import { usePoolRecentProvisions } from "@bera/berajs";

const pool: PoolV2 = ...

function App() {
    const { data: recentProvisions, isLoading } = usePoolRecentProvisions({ pool });

    ...
};
```
