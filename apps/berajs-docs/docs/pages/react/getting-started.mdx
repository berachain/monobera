## Usage

Berajs can be used with any wagmi compatible wallet library. Here is an example of how to use BeraJs with [Dynamic](https://docs.dynamic.xyz/guides/frameworks/wagmi-v2).

```tsx [app.tsx]
import { WagmiProvider } from 'wagmi'
import { config } from './config'
import { BeraJsProvider } from "@bera/berajs";  // [!code focus]
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { DynamicWagmiConnector } from "@dynamic-labs/wagmi-connector";
import { EthereumWalletConnectors } from "@dynamic-labs/ethereum";

function App() {
  const queryClient = new QueryClient();
  return (
    <DynamicContextProvider
        settings={{
          environmentId: "...",
          walletConnectors: [EthereumWalletConnectors],
          overrides: { evmNetworks: ['...'] },
        }}
    >
        <WagmiProvider config={config}>
            <QueryClientProvider client={queryClient}>
                <DynamicWagmiConnector>
                    <BeraJsProvider>  // [!code focus]
                        {/** ... */}   // [!code focus]
                    </BeraJsProvider>  // [!code focus]
                </DynamicWagmiConnector>
          </QueryClientProvider>
        </WagmiProvider>
    </DynamicContextProvider>

  )
}
```

This is just one of many configurations BeraJs can be used.

## Config Override

`BeraJsProvider` by default is configured to point to the contracts and endpoints that the official dapps point to, in order to override these you can use the `configOverride` prop of `BeraJSProvider`. [Learn more about the `configOverride` argument here.](/bera-config)
