# getBeraLpAddress

Helper function to deterministically calculating the contract address for a given pair of tokens.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getBeraLpAddress } from "@bera/berajs/actions";
```

## Usage

```ts [example.ts]
import { getBeraLpAddress } from "@bera/berajs/actions";

const lpTokenAddress = getBeraLpAddress({
  base: "0x...",
  quote: "0x...",
});
```

```ts
import { type GetBeraLpAddress } from "berajs/actions";
```
