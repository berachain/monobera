# getPoolRecentProvisions

Action for retrieving recent provisions data for a given pool. Provisions are pool liquidity add and withdraw events.

## Import

```ts
import { getPoolRecentProvisions } from "@bera/berajs/actions";
```

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Required config

For more information on BeraConfig, see [BeraConfig](/bera-config/)

```ts twoslash
import { type BeraConfig } from "@bera/berajs";

export const beraConfig: BeraConfig = {
  subgraphs: {
    dexSubgraph: "https://...",
  },
};
```

## Usage

```ts
import { getPoolRecentProvisions } from "@bera/berajs/actions";
import { BeraConfig } from "@bera/berajs";

// required config values for getPoolRecentProvisions
export const config: BeraConfig = {
  subgraphs: {
    dexSubgraph: "https://...",
  },
};

const pool: PoolV2 = ...;

const poolRecentProvisions = await getPoolRecentProvisions({
  args: {
    pool
  },
  config
});
```

## Parameters

| Name        | type                              | Description                                                   | Required |
| ----------- | :-------------------------------- | :------------------------------------------------------------ | :------: |
| `args.pool` | `PoolV2`                          | pool of which to fetch recent provisions                      |  `true`  |
| `config`    | [`BeraConfig`](/types/BeraConfig) | BeraConfig object containing addresses for relevant resources |  `true`  |

## Returns

#### Array of the following type `IProvision[]`

See type [`IProvision`](/types/IProvision)
