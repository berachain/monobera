# getBeraLpAddress

Helper function to deterministically calculate the contract address for a given pair of tokens.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getBeraLpAddress } from "@bera/berajs/actions";
```

## Usage

```ts [example.ts]
import { getBeraLpAddress } from "@bera/berajs/actions";

const lpTokenAddress = getBeraLpAddress({
  base: "0x...",
  quote: "0x...",
});
```

## Parameters

| Name         | type      | Description                                                                                  | Required |
| ------------ | :-------- | :------------------------------------------------------------------------------------------- | :------: |
| `args.base`  | `Address` | The onchain address of the base token of the liquidity pool whose address we are retrieving  |  `true`  |
| `args.quote` | `Address` | The onchain address of the quote token of the liquidity pool whose address we are retrieving |  `true`  |

## Returns `string`

The result should be the address of the liquidity pool with the given base and quote token if any.
