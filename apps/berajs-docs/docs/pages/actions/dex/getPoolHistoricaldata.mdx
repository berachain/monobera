# getPoolHistoricalData

Returns a list daily pool history data points up to the past 90 days.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

# Required config

For more information on BeraConfig, see [BeraConfig](/bera-config/)

```ts twoslash
import { type BeraConfig } from "@bera/berajs";

export const beraConfig: BeraConfig = {
  endpoints: {
    dexIndexer: "https://..",
  },
};
```

## Usage

```ts
import { defaultBeraConfig } from "@bera/berajs";
import { getPoolHistoricalData } from "@bera/berajs/actions";

const poolShareAddress = "0x...";

const poolHistoricalData = getPoolHistoricalData({
  shareAddress: poolShareAddress,
  config: defaultBeraConfig,
});
```

## Parameters

- #### `args`

| Name           | type                              | Description                                                   | Required |
| -------------- | :-------------------------------- | :------------------------------------------------------------ | :------: |
| `shareAddress` | `string`                          | The id used to retrieve the pool                              |  `true`  |
| `config`       | [`BeraConfig`](/types/BeraConfig) | BeraConfig object containing addresses for relevant resources |  `true`  |
