# sortBaseQuoteAddresses

Helper function to determine the base and quote tokens of a given two token addresses.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { sortBaseQuoteAddresses } from "@bera/berajs/actions";
```

## Usage

```ts
import { type BeraConfig } from "@bera/berajs";
import { sortBaseQuoteAddresses } from "@bera/berajs/actions";

// required configs for sortBaseQuoteAddresses
export const beraConfig: BeraConfig = {
  contracts: {
    wrappedTokenAddress: "0x...",
  },
};

const [base, quote] = sortBaseQuoteAddresses({
  tokenA: "0x...",
  tokenB: "0x...",
  config: beraConfig,
});
```

## Parameters

| Name          | type                              | Description                                                   | Required |
| ------------- | :-------------------------------- | :------------------------------------------------------------ | :------: |
| `args.tokenA` | `Address`                         | The onchain address of token A                                |  `true`  |
| `args.tokenB` | `Address`                         | The onchain address of token B                                |  `true`  |
| `config`      | [`BeraConfig`](/types/BeraConfig) | BeraConfig object containing addresses for relevant resources |  `true`  |

## Returns `[Address, Address]`
