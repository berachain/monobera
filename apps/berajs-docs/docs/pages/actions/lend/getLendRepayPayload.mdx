# getLendRepayPayload

Action for creating the repay payload for transaction in lend.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started.

## Import

```ts
import { getLendRepayPayload } from "@bera/berajs/actions";
```



## Usage

:::code

```ts [example.ts]
const payload = getLendRepayPayload({ token, amount, account, max: true });
```

:::

## Parameters


| Name     | Type    | Description                               | Required |
| -------- | ------- | ----------------------------------------- | :------: |
| `token`  | `Token` | The token involved in the transaction.    |  `true`  |
| `amount` | `string`| The amount of the token to be transacted. |  `true`  |
| `account`| `Address`| The blockchain address of the user account. |  `true` |
| `max`| `boolean`| Repay all. |  `false` |


## Returns

| Return Type                     | Description                                                        |
| ------------------------------- | ------------------------------------------------------------------ |
| `[Address, bigint, 2, Address]` | An array that can include token address, token amount in bigint, 2, user account address |
