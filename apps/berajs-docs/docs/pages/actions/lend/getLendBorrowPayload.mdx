# getLendBorrowPayload

Action for creating the borrow payload for transaction in lend.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started.

## Import

```ts
import { getLendBorrowPayload } from "@bera/berajs/actions";
```



## Usage

:::code

```ts [example.ts]
const payload  = getLendBorrowPayload({ args: { token, amount, account } });
```

:::

## Parameters


| Name     | Type    | Description                               | Required |
| -------- | ------- | ----------------------------------------- | :------: |
| `token`  | `Token` | The token involved in the transaction.    |  `true`  |
| `amount` | `string`| The amount of the token to be transacted. |  `true`  |
| `account`| `Address`| The blockchain address of the user account. |  `true` |



## Returns

| Return Type                     | Description                                                        |
| ------------------------------- | ------------------------------------------------------------------ |
| `[Address, bigint, 2, 0, Address]` | An array that can include token address, token amount in bigint, 2, 0, user account address |
