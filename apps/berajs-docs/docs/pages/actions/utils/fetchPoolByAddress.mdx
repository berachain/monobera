# getAllowance

Action for fetching the pool by address

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { fetchPoolByAddress } from "@bera/berajs";
```

## Usage

```ts
import { fetchPoolByAddress } from "@bera/berajs";

const poolAddress = "0x...";

const pool: PoolV2 = await fetchPoolByAddress(poolAddress);
```

## Parameters

| Name        | type     | Description                                                        | Required |
| ----------- | :------- | :----------------------------------------------------------------- | :------: |
| poolAddress | `string` | The requested pool's share address on the blockchain ex: ("0x...") |   true   |

## Returns `PoolV2`

See type [`PoolV2`](/types/PoolV2)
