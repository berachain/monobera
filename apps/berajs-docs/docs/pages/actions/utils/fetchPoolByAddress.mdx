# fetchPoolByAddress

Action for fetching a pool's data given the pool's shareAddress (which is the pool's blockchain address)

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

# Required config

For more information on BeraConfig, see [BeraConfig](/bera-config/)

```ts twoslash
import { type BeraConfig } from "@bera/berajs";

export const beraConfig: BeraConfig = {
  endpoints: {
    dexIndexer: "https://..",
  },
};
```

## Import

```ts
import { fetchPoolByAddress } from "@bera/berajs";
```

## Usage

```ts
import { defaultBeraConfig, type PoolV2 } from "@bera/berajs";
import { fetchPoolByAddress } from "@bera/berajs/actions";

const poolAddress = "0x...";

const pool: PoolV2 = await fetchPoolByAddress({
  shareAddress: poolAddress,
  config: defaultBeraConfig,
});
```

## Parameters

| Name           | type                              | Description                                                        | Required |
| -------------- | :-------------------------------- | :----------------------------------------------------------------- | :------: |
| `shareAddress` | `string`                          | The requested pool's share address on the blockchain ex: ("0x...") |  `true`  |
| `config`       | [`BeraConfig`](/types/BeraConfig) | BeraConfig object containing addresses for relevant resources      |  `true`  |

## Returns `PoolV2`

See type [`PoolV2`](/types/PoolV2)
