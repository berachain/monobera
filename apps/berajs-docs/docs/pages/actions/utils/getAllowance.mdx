# getAllowance

Action for fetching the allowance of a given token

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getAllowance } from "@bera/berajs/actions";
```

## Usage

```ts
import { getAllowance } from "@bera/berajs/actions";
import { createPublicClient, http } from "viem";
import { berachain } from "viem/chains";

// first, intialize a public client with berachain
const publicClient = createPublicClient({
  chain: berachain, // viem chains directory: https://github.com/wevm/viem/blob/main/src/chains/index.ts
  transport: http(),
});

const { token, allowance, formattedAllowance } = await getAllowance({
  account,
  token,
  contract,
  publicClient,
});
```

## Parameters
| Name           | type                              | Description                                                               | Required |
| -------------- | :-------------------------------- | :------------------------------------------------------------------------ | :------: |
| tokens       | `Token[]` \| `undefined`     | Array of `Token` objects or undefined                         | true     |
| account      | `` `0x${string}` `` \| `undefined` | Ethereum-style prefixed blockchain account address or undefined | true     |
| config       | `BeraConfig`                 | Configuration settings specific to the allowance retrieval    | false    |
| spender      | `string` \| `undefined`      | Identifier for the entity allowed to spend or undefined       | true     |
| publicClient | `PublicClient` \| `undefined`| Client interface for public blockchain interactions or undefined | true     |

## Returns

#### [`Promise<AllowanceToken[] | undefined>`](/types/AllowanceToken/)
