# getBeraBalance

Action for fetching the current bera balance of a given address

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getBeraBalance } from "@bera/berajs/actions";
```

## Usage

```ts
import { getBeraBalance } from "@bera/berajs/actions";

const exampleAddress = "0x1..";

const data = await getBeraBalance({
  address: exampleAddress,
  publicClient,
});
```


## Parameters

| Name           | type                              | Description                                                               | Required |
| -------------- | :-------------------------------- | :------------------------------------------------------------------------ | :------: |
| account      | `string` \| `undefined`   | Blockchain account address or undefined                  | true     |
| publicClient | `PublicClient` \| `undefined` | Client interface for public blockchain interactions or undefined | true     |

## Returns

#### [`Promise<TokenBalance | undefined>`](/types/TokenBalance/)
