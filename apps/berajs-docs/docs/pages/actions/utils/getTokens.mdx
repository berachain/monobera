# getTokens

Action for fetching and formatting the token list

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getTokens } from "@bera/berajs/actions";
```

## Usage

```ts
import { type Token } from "@bera/berajs";
import { getTokens } from "@bera/berajs/actions";

import { beraConfig } from "./config";

const customToken: Token = {
  address: "0x0...",
  decimals: 18,
  symbol: "TKN",
  name: "TKN",
};

const tokens = await getTokens({
  externalList: [customToken], // optional list of Token Objects to combine with the token list in the config
  config: beraConfig,
});
```

## Parameters

| Name         | type         | Description                                                    | Required |
| ------------ | :----------- | :------------------------------------------------------------- | :------: |
| externalList | `Token[]`    | Optional array of `Token` objects from an external source      |   true   |
| config       | `BeraConfig` | Configuration settings specific to the token retrieval request |  false   |
