# getWalletBalances

Action for fetching the balances of a given wallet address

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getWalletBalances } from "@bera/berajs/actions";
```

## Required config

For more information on BeraConfig, see [BeraConfig](/bera-config/)

```ts twoslash
import { type BeraConfig } from "@bera/berajs";

export const beraConfig: BeraConfig = {
  endpoints: {
    tokenList: "https://..",
  },
};
```

## Usage

```ts
import { type BeraConfig, type Token } from "@bera/berajs";
import { getWalletBalances } from "@bera/berajs/actions";
import { createPublicClient, http } from "viem";
import { berachain } from "viem/chains";

// first, intialize a public client with berachain
const publicClient = createPublicClient({
  chain: berachain, // viem chains directory: https://github.com/wevm/viem/blob/main/src/chains/index.ts
  transport: http(),
});

// required configs for getWalletBalances
export const beraConfig: BeraConfig = {
  endpoints: {
    tokenList: "https://..",
  },
};

const walletBalances = await getWalletBalances({
  account,
  tokenList,
  config: beraConfig,
  publicClient,
});
```

## Parameters

| Name         | type                          | Description                                               | Required |
| ------------ | :---------------------------- | :-------------------------------------------------------- | :------: |
| account      | `string` \| `undefined`       | Blockchain account address                                |   true   |
| tokenList    | `Token[]` \| `undefined`      | Array of `Token` objects representing the wallet's tokens |   true   |
| config       | `BeraConfig`                  | Configuration settings for retrieving wallet balances     |  false   |
| publicClient | `PublicClient` \| `undefined` | Client interface for public blockchain interactions       |   true   |
