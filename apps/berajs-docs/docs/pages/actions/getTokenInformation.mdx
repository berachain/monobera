# getTokenInformation

Action for fetching token information

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getTokenInformation } from "@bera/berajs";
```

## Usage

:::code-group

```ts twoslash [example.ts]
import { defaultBeraConfig, getTokenInformation } from "@bera/berajs";
import { useConfig, usePublicClient } from "wagmi";

const config = useConfig();

await getTokenInformation({
  dispatch,
  address,
  setTokenInformation,
  setError,
  config, // wagmi config
});
```
