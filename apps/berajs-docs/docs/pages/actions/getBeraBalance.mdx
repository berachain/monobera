# getBeraBalance

Action for fetching the current bera balance of a given address

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getBeraBalance } from "@bera/berajs";
```

## Usage

:::code-group

```ts [example.ts]
import { getBeraBalance } from "@bera/berajs";

const exampleAddress = "0x1..";

const data = await getBeraBalance({
  address: exampleAddress,
  publicClient,
});
```

````

:::

## Returns

```ts
Promise<number | undefined>
````
