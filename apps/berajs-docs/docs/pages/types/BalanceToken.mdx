## `BalanceToken`

| Name     | Type       | Description                                                      | Optional |
|----------|------------|------------------------------------------------------------------|----------|
| logoURI  | `string`   | URL of the token's logo image                                    | true     |
| default  | `boolean`  | Indicates if the token is a default option (to be deprecated)    | true     |
| tags     | `string[]` | Array of tags associated with the token                          | true     |
| address  | `Address`  | Blockchain address of the token                                  | false    |
| decimals | `number`   | Number of decimals the token uses                                | false    |
| symbol   | `string`   | Symbol representing the token                                    | false    |
| name     | `string`   | Full name of the token                                           | false    |
| balance          | `bigint`   | Numeric balance value using bigint for precision   | false    |
| formattedBalance | `string`   | Human-readable string format of the balance        | false    |
