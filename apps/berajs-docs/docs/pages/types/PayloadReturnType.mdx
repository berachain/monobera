## `PayloadReturnType<T>`

| Name      | type     | Description                                                                         | Optional |
| --------- | :------- | :---------------------------------------------------------------------------------- | :------: |
| `payload` | `T`      | arguments to be sent in a blockchain transaction payload                            | `false`  |
| `value`   | `bigint` | value to be sent in a blockchain transaction, this refers to msg.sender in solidity |  `true`  |
