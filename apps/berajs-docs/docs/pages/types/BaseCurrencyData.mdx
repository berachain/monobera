## `BaseCurrencyData`

| Name                                   | Type     | Description                                                                | Optional |
| -------------------------------------- | -------- | -------------------------------------------------------------------------- | :------: |
| `marketReferenceCurrencyDecimals`      | `number` | Number of decimals used for the market reference currency. default 8.                 |  `false` |
| `marketReferenceCurrencyPriceInUsd`    | `string` | Price of the market reference currency in US dollars.                      |  `false` |
| `networkBaseTokenPriceInUsd`           | `string` | Price of the network's base token in US dollars.                           |  `false` |
| `networkBaseTokenPriceDecimals`        | `number` | Number of decimals used for pricing the network's base token. default 18.        |  `false` |
