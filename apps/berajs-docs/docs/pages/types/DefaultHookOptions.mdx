## `DefaultHookOptions`

| Name     | type                              | Description                                                   | Required |
| -------- | :-------------------------------- | :------------------------------------------------------------ | :------: |
| `config` | [`BeraConfig`](/types/BeraConfig) | BeraConfig object containing addresses for relevant resources | `false`  |
| `opts`   | `SWRConfiguration`                | SWR options for the underlying useSWR hook                    | `false`  |
