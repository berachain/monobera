## `ICrocSwapStep`

| Name    | Type      | Description                                                                         | Optional |
| ------- | --------- | ----------------------------------------------------------------------------------- | -------- |
| poolIdx | `bigint`  | the pool on which the swap step will operate on's on-chain id                       | false    |
| base    | `Address` | the pool's base token address                                                       | false    |
| quote   | `Address` | the pool's quote token address                                                      | false    |
| isBuy   | `boolean` | true when the swap is base to quote token, false if the swap is quote to base token | false    |
