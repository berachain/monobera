## `GetTokens`

| Name                   | type                                                             | Description                                                                     | Optional |
| ---------------------- | :--------------------------------------------------------------- | :------------------------------------------------------------------------------ | :------: |
| tokenList          | `Token[]` \| `undefined`             | Array of `Token` objects                               | true     |
| customTokenList    | `Token[]` \| `undefined`             | Array of custom `Token` objects                        | true     |
| tokenDictionary    | `{ [key: string]: Token }` \| `undefined` | Dictionary of `Token` objects keyed by a string        | true     |
| featuredTokenList  | `Token[]` \| `undefined`             | Array of featured `Token` objects                      | true     |

See type [`Token`](/types/Token/)
